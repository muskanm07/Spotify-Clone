# Spotify Clone

A simple **Spotify Clone** web application built using **HTML, CSS, and Bootstrap**. This project mimics the basic look and feel of Spotify and allows users to explore a music-style interface.

## Features
- Responsive layout with Bootstrap
- Homepage with music playlists
- Search bar for songs (UI only)
- Music cards with album art, song title, and artist
- Navigation bar similar to Spotify
